# System Requirements

## Option 1: Docker Desktop (Recommended - Easiest)
- Docker Desktop 4.0+ 
  - Windows 10/11 Pro/Enterprise, or Windows 10 Home with WSL2
  - macOS 10.14+
  - Linux (Ubuntu 20.04+, Debian, Fedora, etc.)
- Disk space: ~1GB for Docker image

## Option 2: Local Node.js Development
- Node.js 20 LTS or higher
- npm 10+ (included with Node.js)
- Disk space: ~500MB for node_modules

## Recommended System Specs
- RAM: 2GB minimum (4GB+ recommended)
- CPU: Dual-core processor
- Internet: Required for downloading dependencies and running scans

## Installation Sources
- Docker Desktop: https://www.docker.com/products/docker-desktop
- Node.js: https://nodejs.org/ (use LTS version)

## Optional
- Git (for cloning repository)
- Code editor (VS Code, WebStorm, etc.)
